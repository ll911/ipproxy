ipproxy

Proxy to zero the last octet of the ip-address from Google Analytics client requests to /collect and then pass them along to Analytics using the Google Analytics Measurement Protocol.

Mainly useful if you prefer to do ip-address anonymization yourself (e.g. to conform with local legal requirements).

For this to have any purpose, you need to prevent analytics.js hitting Google, and make it hit this proxy instead (e.g. using tasks)
